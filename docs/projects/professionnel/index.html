Professionnel

attention

Seuls les projets dont les données sont publiques sont présentés dans cette section. Les réalisations soumises à des restrictions de confidentialité ne figurent pas ici. Toutes les informations affichées sont accessibles via les canaux publics de l’employeur (rapports d’activité, site web, réseaux sociaux, etc.).

Cette section regroupe des projets réalisés dans le cadre professionnel. Les exemples présentés concernent des missions ou réalisations rendues publiques par l’employeur.

Types de projets :

  • Développement et intégration de systèmes robotiques complexes
  • Mise en œuvre de solutions d’intelligence artificielle et de vision par ordinateur
  • Navigation autonome, manipulation d’objets et interaction homme-machine
  • Coordination d’équipes pluridisciplinaires et participation à des compétitions internationales

📄️BlueNav - Jumeau numérique bateau autonome

Jumeau numérique pour un bateau autonome de mesure hydrographique. Simulation ROS2, visualisation temps réel et CI/CD embarqué pour le projet BlueNav au CATIE.

📄️CI/CD - Workflows GitHub Actions mutualisés

Bibliothèque de workflows GitHub Actions réutilisables pour Python, Docker et Helm. Déploiement standardisé sur cluster Kubernetes pour toute l’organisation CATIE.

📄️GitHub ARC - Runners CI/CD auto-hébergés sur Kubernetes

Déploiement de GitHub Actions Runner Controller (ARC) sur un cluster Kubernetes kubeadm. Runners éphémères dans des pods isolés, secrets chiffrés, accès registre privé.

📄️Inspection visuelle automatisée de véhicules

Pipeline edge de vision par ordinateur pour la détection automatique de dommages sur véhicules neufs en parcs portuaires. Inférence sur NPU embarqué, traitement RAW en temps réel.

📄️Automatisation du quotidien — outils internes SONU

Bots Slack, dashboard Jira, alertes ERP et site de documentation interne. Outils développés et déployés sur Kubernetes pour automatiser les tâches répétitives de l'équipe SONU au CATIE.

📄️RoboCup @Home 2023 - CATIE Robotics

3ème place mondiale à la RoboCup @Home 2023 à Bordeaux. Contribution sur l'architecture logicielle ROS2 et la navigation autonome du robot Reachy.

📄️Cluster Kubernetes interne SONU

Cluster Kubernetes bare-metal monté avec kubeadm sur 7 nœuds. GitOps via Helm, exposition Tailscale, chaîne d’observabilité Prometheus/Grafana/Loki. Infrastructure interne de l’équipe SONU au CATIE.

📄️Standards Python - Template Cookiecutter

Template Cookiecutter Python normalisé pour l'organisation CATIE. Poetry, pre-commit, pytest, CI GitHub Actions et documentation automatique.
