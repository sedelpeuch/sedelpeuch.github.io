Cycle Préparatoire De Bordeaux - Semestre 4

Informatique​

Responsable Frantisek Kardos

Au semestre 4 le cours d'informatique propose de mettre en place un jeu de votre choix (dont le thème change chaque année !). Pour ces qui sont les plus intéressé (souvent les ENSEIRB-Info) c'est une occasion de réaliser un projet complet en informatique. Ne foncez surtout pas tête baissée, prenez le temps de réflechir à comment va s'organiser les fichiers, les fonctions etc. Amusez vous !

Au semestre 4 le projet est un élément central. Ils peuvent totalement sauver le semestre car ils fournissent des notes en LCO, Anglais mais aussi dans l'UE projet (coefficient 6). Voici quelques exemples de projet qui ont marché les années précendentes, vous pouvez les parcourir pour voir ce qui est attendu ou juste par curiosité. Les différents professeurs ont différentes attentes qu'il faut respecter.
