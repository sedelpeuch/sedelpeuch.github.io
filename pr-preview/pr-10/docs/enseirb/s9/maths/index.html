Mathématiques pour la robotique

À partir d'une position initiale, le robot tourne de θ\thetaθ puis avance de ttt puis tourne de α\alphaα puis avance de ddd. Donner son positionnement dans le repère (x,y,θ)(x,y,\theta)(x,y,θ).

  • Le robot tourne de θ\thetaθ puis avance de ttt. Dans le repère ΩO=(O,i⃗,j⃗)\Omega_O = (O, \vec{i}, \vec{j})ΩO​=(O,i,j​) :
T⃗ΩO=(uv)=t.(cos⁡θsin⁡θ)\vec{T}_{\Omega_O} = \begin{pmatrix} u\\ v \end{pmatrix} = t. \begin{pmatrix} \cos{\theta} \\ \sin{\theta} \end{pmatrix}TΩO​​=(uv​)=t.(cosθsinθ​)
  • Puis, le robot tourne de α\alphaα puis avance de ddd. Dans le repère ΩC=(C,x⃗,y⃗)\Omega_C = (C, \vec{x}, \vec{y})ΩC​=(C,x,y​) :
D⃗ΩC=(pq)=d.(cos⁡αsin⁡α)\vec{D}_{\Omega_C} = \begin{pmatrix} p\\ q \end{pmatrix} = d. \begin{pmatrix} \cos{\alpha} \\ \sin{\alpha} \end{pmatrix}DΩC​​=(pq​)=d.(cosαsinα​)

Modèles directe / inverse​

  • Dans quel espace ? : Le mécanisme à 3 degrés de liberté
  • Quels sont les variables ? : Les entrées vont être le tuple de θ\thetaθ ainsi que les longueurs des 3 paramètres ttt
  • **Poser le problème ** : Dans le modèle directe on souhaite trouver les coordonées cartésiennes en fonction des θ\thetaθ et des longueurs ttt. Le modèle inverse lui fournit la position des différents joints θ\thetaθ en fonction de la position cartésienne de l'effecteur.
