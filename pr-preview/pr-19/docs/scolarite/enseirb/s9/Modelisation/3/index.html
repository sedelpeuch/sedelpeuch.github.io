Modèle cinématique

La cinématique est l'étude des vitesses d'un point lié à un repère (vitesse de translation, de rotation). Pour cela nous allons utiliser la jacobienne d'une articulation en un point.

Définition : un repère est lié à un solide si pour tout point PPP du solide est lié au repère c'est à dire que les coordonnées de PPP sont fixes dans ce repère.

Définition : la vitesse du point PPP par rapport au repère RiR_iRi​ est le vecteur dont les coordonnées dans le repère iii sont les dérivées temporelles des coorodonnées du point PPP dans le repère iii. ivP/Ri=diPdt{}^i v_{P/R_i} = \frac{d{}^i P}{dt}ivP/Ri​​=dtdiP​
